### 교환 정렬 ⭐️

- [버블 정렬 `bubble_sort.ts`](./ExchangeSort/bubble_sort.ts)

  | 복잡도          | 수학식                       |
  | --------------- | ---------------------------- |
  | 최악 시간복잡도 | $O(n^2)$ 비교, $O(n^2)$ 교환 |
  | 최선 시간복잡도 | $O(n)$ 비교, $O(1)$ 교환     |
  | 평균 시간복잡도 | $O(n^2)$ 비교, $O(n^2)$ 교환 |
  | 공간복잡도      | $O(1)$ 보조                  |
